# /camp (Legacy)

{% hint style="danger" %}
This command was removed in v3.0.0.
{% endhint %}

The `/camp` command is used to create safezone plots, destroy them, and to give or take plots to or from other players.

### Command Structure

`/safezone <parameter>`

_**Valid**_**&#x20;Parameters**

* build
* destroy
* help
* info
* plots
  * give
  * take
